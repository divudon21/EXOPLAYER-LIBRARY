# QUICK PLAY
A simple and powerful IPTV and media player app.

## Features
- **M3U Playlist Support:** Add custom playlists with specific User-Agents
- **DRM Support:** Widevine, PlayReady, ClearKey via M3U Kodi tags
- **External Player:** Open video links and local files directly from browsers or managers like 1DM
- **Immersive Fullscreen:** True edge-to-edge video playback
- **Quality & Subtitle Selection:** Built-in ExoPlayer track selector

## Build Instructions
To build the app yourself, run:
```bash
./gradlew assembleDebug
```
The APK will be located at `app/build/outputs/apk/debug/app-debug.apk`.

Alternatively, check the **Actions** tab on GitHub to download the latest automated build.
